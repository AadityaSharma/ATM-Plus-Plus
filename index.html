ATM++

Voice User Interface (VUI) with Emotions for an ATM Machine

It's a voice user interface enabled application for an ATM Machine. HTML5 Web Speech API is used for Text-To-Speech and Speech-To-Text Conversion. The goal of this project is to make ATM Machines accessible for blind and handicapped users.
